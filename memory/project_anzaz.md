ANZAZ is a fully-frontend React Native Expo Go app built from low-fidelity wireframes.

**Why:** Demo/portfolio project with no backend; all data is hardcoded in `src/data/mockData.ts`.

**How to apply:** When the user asks to add features, keep them frontend-only. If backend integration is requested later, the mock data file is the natural seam to replace.

Tech: Expo SDK 51, React Navigation 6 (Stack + Bottom Tabs), Reanimated 3, Lucide React Native, TypeScript.

Screens: Splash, Auth, Home, Capture (scan animation), Result (species ID), Map (custom illustrated map with animated pins), Journal (grid/list toggle), Profile (badges + leaderboard), SpeciesDetail.

Design: primary #2d8a4e, background #f5f5f5, Inter/system font, no emojis.
